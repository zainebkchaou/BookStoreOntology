# BookStore Ontology
This project represents an ontology for an online bookstore system.
## Domain
Books, customers, categories, shopping cart, orders and payments.
## Tools Used
- Protégé
- OWL / RDF
- GraphDB
- GitHub
- RDF
- SPARL
## Setup
1. Open Protégé
2. Load ontology.owl
3. Import into GraphDB
4. Run SPARQL queries
